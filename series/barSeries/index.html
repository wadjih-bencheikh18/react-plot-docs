Like all other series, scatter series have Base props
They also have many other props that we can class into the following categories :

1. Markers‚Äã

  • displayMarkers: display scatter points markers
    type: boolean
    default: true
  • markerShape: serie's markers shape
    type: Shape
    default: 'circle'
  • markerSize: serie's markers size
    type: number
    default: 8
  • markerStyle: serie's markers style that can be dynamique or statique
    type: CSSFuncProps
    default: {}

2. Point Label‚Äã

  • pointLabel: serie's point label that can be statique or dynamique depending points
    type: LabelFuncProps<point>
    default: ''
  • pointLabelStyle: serie's point label style, statique or dynamique depending points
    type: CSSFuncProps<point>
    default: {}

3. ErrorBars‚Äã

  • displayErrorBars: display errors bars in the serie
    type: boolean
    default: false
  • errorBarsStyle: change error bars style
    type: SVGAttributes<SVGLineElement>
  • errorBarsCapSize: size of error bars cap
    type: number
  • errorBarsCapStyle: change error bars cap's style
    type: SVGAttributes<SVGLineElement>

4. Lines‚Äã

  • lineStyle: styling of the serie's path statique or dynamique that depend series id
    type: CSSFuncProps<{id}>
    default: false
